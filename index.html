Cool stuff

This is a list of things I think are cool, either that I've made or someone else has.
  • Pluto: An x86 OS kernel a friend and I have been working on since 2019. We have amd64 and AArch64 ports in the works (they might even be finished by time you're reading this)
  • Zig: A really cool systems programming language. It's like C but better, basically. It has some nice compile-time code execution, error handling mechanics and supports freestanding code better than anything I've ever seen. Check it out!
  • Serenity: A hobby OS that tries to recapture the allures of 90s and early 2000s Windows and merge them with the functionality of Unix
  • The Architecture of Open Source Applications: A great website that describes the architecture of many applications (CI, template engines, python interpreters etc.) by guiding the reader through writing them
  • Aave: Aave allows you to deposit your crypto assets and borrow other assets against them. The power to turn your idle assets into something you can use for other purposes, without selling them, is so powerful
